# v0.1.3 Release Notes

**Date:** 2026-04-03

## Fixes

### Gitignore local-only glob and cleanup

Fixed the `local-only-*` gitignore pattern to use `shared-context/**/local-only-*` so it covers user-scoped directories (e.g., `shared-context/myron/local-only-*`). The previous pattern only matched root-level files, meaning local-only braindumps in user dirs showed up as untracked. Removed `.superpowers/` from gitignore so plugin artifacts get flagged by workflow for proper placement rather than being silently ignored. Added `.idea/` for JetBrains IDEs.
