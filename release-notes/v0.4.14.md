# v0.4.14 Release Notes

**Date:** 2026-04-07

## Fixes

The session-start hook now captures the Claude Code session ID (UUID) from its stdin input and writes it to `.claude-scratchpad/.current-chat-id`. Skills read the chat ID from this file instead of guessing from `.jsonl` file timestamps. Added `getCurrentChatId()` helper to `_utils.mjs`. Fallback to file listing if the scratchpad file doesn't exist (hook didn't fire).
