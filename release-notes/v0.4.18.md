# v0.4.18 Release Notes

**Date:** 2026-04-10

## Features

New optional rule: `local-dev-environment.md.skip`. When activated, Claude maintains a `shared-context/locked/local-only-dev-environment.md` file that captures machine-specific development context — services & ports, credentials, CLI commands, env vars, paths, infrastructure details. The file is gitignored (local-only prefix) but positioned in locked for maximum attention. Claude records details proactively as they're discovered during work, asking for confirmation only before saving credentials.
