# v0.4.3 Release Notes

**Date:** 2026-04-06

## Fixes

`--init` now replaces the existing CLAUDE.md with a clean bootstrap version instead of appending to it. The old CLAUDE.md is backed up to `CLAUDE.md.bak`. Previously, appending the workspace-update @-references to an existing CLAUDE.md meant old instructions (e.g., Notion MCP rules) would take priority and prevent the workspace initialization hook from driving the setup flow.
