# v0.4.7 Release Notes

**Date:** 2026-04-06

## Fixes

`--init` now installs only the 3 bootstrap skills (setup, workspace-init, workspace-update) plus hooks and scripts. Previously installed the entire `.claude/` directory which could overwrite existing user configuration. The remaining skills, agents, and rules are applied through `/setup` or `/workspace-update` where the user has control over what gets installed.

Existing files are never overwritten — only missing files are copied.
