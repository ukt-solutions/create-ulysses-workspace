# v0.4.8 Release Notes

**Date:** 2026-04-06

## Fixes

Merged `/setup` into `/workspace-init`. One skill, one post-scaffold flow. `/workspace-init` now handles everything: clone repos, install template components, activate optional rules, configure user identity, extract team knowledge from documentation sources, populate shared context, and clean up.

The `/setup` skill has been removed. All references updated across docs, guides, README, CLAUDE.md template, and the CLI output.

The init CLI now installs only 2 bootstrap skills (workspace-init, workspace-update) instead of 3.
